# Shoe Sales Analysis Dashboard (Excel)

## Project Overview
Developed an interactive sales dashboard using Microsoft Excel to analyze shoe sales performance across different regions, products, and time periods.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Functions

## Key Features
- Sales Trend Analysis
- Region-wise Sales
- Product Performance
- Monthly Sales Analysis
- Interactive Dashboard

## Key Insights
- Identified top-performing products.
- Compared regional sales performance.
- Analyzed monthly sales trends.
- Built an interactive dashboard for business reporting.
